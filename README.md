# CProgram-kntu-404
توضیح کلی پروژه
این پروژه یک برنامه‌ی ساده شبیه صفحه‌گسترده (Excel) است که با زبان C پیاده‌سازی شده است.
برنامه از یک جدول دوبعدی (Sheet) تشکیل شده که هر خانه (Cell) می‌تواند شامل عدد، متن یا فرمول باشد.
هدف پروژه، تمرین مفاهیم مدیریت حافظه، کار با فایل، پارس کردن عبارات ریاضی و ارجاع بین سلول‌ها است.

قابلیت‌های اصلی برنامه

ایجاد جدول با تعداد سطر و ستون دلخواه

ذخیره و بارگذاری جدول از فایل CSV

پشتیبانی از فرمول‌ها (شروع با =)

امکان استفاده از عملگرهای ریاضی + - * / ^

پشتیبانی از توابع ریاضی مثل:
sin , cos , tan , cot , sqrt , log , ln , exp , abs

امکان ارجاع به سلول‌ها در فرمول‌ها (مثل A1 ، B2 و ...)

بررسی خطاهای رایج مثل:

تقسیم بر صفر

دامنه‌ی نامعتبر توابع

فرمول اشتباه یا پرانتز نامتعادل

نکته‌ی مهم درباره توابع مثل sin و cos

توابع ریاضی در زبان C (کتابخانه math.h) با رادیان کار می‌کنند، نه درجه.

مثال:
sin(60) ← این یعنی سینوس ۶۰ رادیان
به همین دلیل مقدار آن حدوداً برابر است با:
-0.3048

اگر منظور شما سینوس ۶۰ درجه است، باید بنویسید:
sin(60 * PI / 180)

ساخت (کامپایل) برنامه

برای کامپایل برنامه از Makefile استفاده شده است.

در پوشه‌ی اصلی پروژه دستور زیر را اجرا کنید:

mingw32-make

بعد از کامپایل، فایل اجرایی با نام زیر ساخته می‌شود:
Exel_Program.exe

فرمت فایل CSV

هر سطر فایل نماینده‌ی یک سطر جدول است

ستون‌ها با کاما (,) جدا می‌شوند

اگر مقدار یک سلول فرمول باشد، باید با = شروع شود

ساختار پوشه‌های پروژه

پوشه Cell-Sheet
شامل کدهای مربوط به ساخت، تغییر اندازه، چاپ، اعتبارسنجی و آزادسازی جدول و سلول‌ها

پوشه File
شامل کدهای مربوط به ذخیره و بارگذاری فایل CSV و بررسی فرمت فایل

پوشه Formula
شامل کدهای مربوط به پردازش فرمول‌ها، پارس عبارات و محاسبه‌ی نتیجه

فایل main.c
نقطه‌ی شروع برنامه و منوی اصلی

فایل Makefile
برای کامپایل خودکار کل پروژه
